# 🎓 HTML CSS Bootstrap Project 9

A modern and responsive **IT Training Academy Landing Page** built using **HTML5, CSS3, and Bootstrap 5**. This project showcases an educational institute website with a clean user interface, responsive design, featured courses, success statistics, student testimonials, and a professional footer.
